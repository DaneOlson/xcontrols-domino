Bootstrap Controls is an open source project created by @mattwhite, @sives and @jackherbert and sponsored by Teamstudio. It is based upon the Bootcards project.

It was originally released in 2012 and has undergone several incarnations since then. It was created to meet Teamstudio's need for a modular user interface framework that could be used to build enterprise mobile apps very quickly using IBM XPages. By building on top of Bootstrap, Bootcards takes advantage of Bootstrap's great capabilities, whilst adding support for the Cards design pattern.
